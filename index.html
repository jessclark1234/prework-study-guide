HTML
  • The head element contains information about the webpage
  • HTML stands for HyperText Markup Language
  • The body element represents th visible content shown to user
  • HTML gives content structure
  • HTML is one of the three cornerstone languages in web development
  • Uses text, image, header, and list elements to provide content and basic function to all webpages
  • HTML also uses meta-related, content division, and semantic elements to divide and describe content
  • Two most important elements for HTML are head and body [left off angle brackets to not wreck code]
  • HTML elements define content and framework
  • HTML can also be used to divide content on page as well as describe content to other developers/browsers
CSS
  • A margin indicates how much space we want around the outside of the element
  • CSS stands for Cascading Stylesheet
  • A padding indicates how much space we want around the content inside of an element
  • CSS defines how webpage content is presented to users
  • There are three types of CSS: inline, internal style sheet, and external style sheet
  • External style sheet is the most common and preferred as it keeps all CSS rules in seperate file
  • CSS is one of the three cornerstone languages in web development
  • Declarations are used to specify which element's properties you want to style
  • Properties are ways you can style an HTML element
  • Each declaration must use a color to seperate the property from the value
Git
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates new branch and switches to it
  • git add -A: adds modification in current working branch to staging area
  • git pull origin main: pushes worked on code into Github repository
  • git log: show all commits in current branch
  • git reset: unstage file while retaining changes in working directory
  • git branch: lists branches with * appearing next to branch in use
  • git add [file]: add a file as it looks now next to commit
  • git merge [branch]: merges specified branch's history into current one
  • git stash: save modifies and staged changes
JavaScript
  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • Console.log method is used to output a message to the web console
  • One of the three cornerstone languages in web development
  • Used to add interactivity to webpage
  • JavaScript was invented by Brendan Eich
  • Can also be used to create video games
  • JavaScript is compact and flexible
  • JavaScript is versatlie and beginner-friendly
  • JavaScript is case sensitive
