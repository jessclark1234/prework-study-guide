HTML
  • The head element contains information about the webpage
  • The body element represents th visible content shown to user
  • HTML gives content structure
  • HTML is one of the three cornerstone languages in web development
  • Uses text, image, header, and list elements to provide content and basic function to all webpages
  • HTML also uses meta-related, content division, and semantic elements to divide and describe content
CSS
  • A margin indicates how much space we want around the outside of the element
  • A padding indicates how much space we want around the content inside of an element
  • CSS defines how webpage content is presented to users
  • There are three types of CSS: inline, internal style sheet, and external style sheet
  • External style sheet is the most common and preferred as it keeps all CSS rules in seperate file
  • CSS is one of the three cornerstone languages in web development
Git
  • git status:checks what branch we are currently on
  • git checkout -b branch-name:creates new branch and switches to it
  • git add -A:adds modification in current working branch to staging area
  • git pull origin main: pushes worked on code into Github repository
